Categorización de los Servicios de Transformación y Digital

Servicios Gold

Un servicio es GOLD si responde a una o más de las siguientes afirmaciones:

Cualquier sistema que se considere crítico para DEACERO al soportar procesos Make, Move, Sell, Pay.
La interrupción o cualquier pérdida de datos asociada con este sistema podría causar un gran impacto económico, o implicaciones legales.
El sistema es utilizado en varias ubicaciones .
Es necesario que el sistema funcione 24/7.

Servicios Silver

Un servicio es Silver si responde a una o más de las siguientes afirmaciones:

Cualquier sistema cuya interrupción o pérdida de datos causaría un impacto significativo de manera inmediata.
El sistema es utilizado en una única ubicación.
Es necesario garantizar que el sistema funcione dentro del horario normal de oficina (L-V 8:30 a 6:30 pm) y se espera que esté disponible fuera del horario laboral normal.

Servicios Bronze

Un servicio es Bronze si responde a una o más de las siguientes afirmaciones:

Cualquier sistema cuya interrupción o pérdida de datos causaría un impacto, pero no inmediato, ni significativo.
El sistema es utilizado en una única ubicación.
Es necesario garantizar que el sistema funcione dentro del horario normal de oficina (L-V 8:30 a 6:30 pm) y no se espera que esté disponible fuera del horario laboral normal.
